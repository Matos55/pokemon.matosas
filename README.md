Tiny local directory for associating community avatar names with faces.

Run locally
- Open `C:/Users/Joao_/Desktop/Trainning/Github/PokemonMatosas/index.html` in your browser.

Edit data
- Edit `people.json` to add or update entries. Each entry has `name`, `avatar`, and `image` (URL).

Example entry:

```
{
  "name": "Joao",
  "avatar": "bakemyday",
  "image": "https://i.pravatar.cc/150?u=joao"
}
```

Search
- Type a real name (e.g. "Joao") or avatar (e.g. "bakemyday") into the search box to filter results.
